# Case-file-summarization
A Python-based NLP pipeline for summarizing legal case documents.

## Workflow

1. Split PDF into manageable chunks.
2. Perform OCR and combine extracted text.
3. Generate summaries using the BART model.

## Technologies

- Python
- PyMuPDF
- OCR
- Hugging Face Transformers
- BART
- NLP

## Usage

python pdfsplit.py
python ocr_combiner.py
python bartbase.py
